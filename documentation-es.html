Documentación de Funciones del Juego s-found 2

Este documento describe un conjunto de funciones JavaScript utilizadas en un juego basado en s-found 2 para la gestión de sprites, detección de colisiones, movimiento y otras mecánicas del juego.

onSpriteClick(sprite, pointer)

Parámetros:

  • sprite: El objeto sprite en el que se hizo clic.
  • pointer: El objeto puntero que representa el evento de clic.

Descripción: Un controlador de eventos predeterminado para eventos de clic en sprites. Puede ser sobrescrito para implementar un comportamiento personalizado.

RadianstoDegrees(angle)

Parámetros:

  • angle: Ángulo en radianes.

Descripción: Convierte un ángulo de radianes a grados.

Retorna: El ángulo en grados.

DegreestoRadians(angle)

Parámetros:

  • angle: Ángulo en grados.

Descripción: Convierte un ángulo de grados a radianes.

Retorna: El ángulo en radianes.

normalize_angle_hack(angle)

Parámetros:

  • angle: Ángulo en grados a normalizar.

Descripción: Normaliza un ángulo al rango [0, 360) grados usando una operación de módulo repetitiva.

Retorna: El ángulo normalizado en grados.

clone_sprite()

Parámetros: Ninguno

Descripción: Crea un clon del sprite actual, copiando su posición, clave, marco y marcos. El clon se agrega a la capa frontal, se escala y se habilita con física con rotación centrada. El sprite original se oculta para evitar duplicación.

destroy_current_sprite()

Parámetros: Ninguno

Descripción: Destruye el sprite actual y sus marcos. Si el sprite es un clon, elimina el clon de la lista de clones del sprite.

hide_current_sprite()

Parámetros: Ninguno

Descripción: Oculta el sprite actual y sus marcos estableciendo su visibilidad en falso. Maneja tanto sprites clonados como no clonados.

show_current_sprite()

Parámetros: Ninguno

Descripción: Muestra el sprite actual y su marco actual estableciendo su visibilidad en verdadero. Maneja tanto sprites clonados como no clonados.

find_sprite_object_by_name_new(sprite_name)

Parámetros:

  • sprite_name: Nombre del sprite a encontrar.

Descripción: Similar a find_sprite_object_by_name, encuentra un sprite por su nombre, manejando clones si aplica.

Retorna: El objeto sprite o null si no se encuentra o si no se proporciona un nombre de sprite.

find_target_sprite_with_clones_object_by_name(sprite_name)

Parámetros:

  • sprite_name: Nombre del sprite objetivo a encontrar.

Descripción: Encuentra un sprite objetivo por su nombre, retornando un clon si se especifica un clon objetivo mediante current_target_clone.

Retorna: El objeto sprite o null si no se encuentra o si no se proporciona un nombre de sprite.

set_current_sprite_name(sprite_name)

Parámetros:

  • sprite_name: Nombre del sprite a establecer como actual.

Descripción: Establece la variable global current_sprite_name al nombre de sprite proporcionado.

set_target_sprite_name(sprite_name)

Parámetros:

  • sprite_name: Nombre del sprite a establecer como objetivo.

Descripción: Establece la variable global target_sprite_name al nombre de sprite proporcionado.

point_in_direction_degrees(angle_degrees)

Parámetros:

  • angle_degrees: Ángulo en grados para rotar el sprite.

Descripción: Rota el sprite actual para que mire hacia el ángulo especificado (en grados).

point_sprite_in_direction_to_point(destination_x, destination_y)

Parámetros:

  • destination_x: Coordenada X del punto objetivo.
  • destination_y: Coordenada Y del punto objetivo.

Descripción: Rota el sprite actual para que mire hacia un punto especificado en el mundo del juego.

point_in_direction(x1, y1, x2, y2)

Parámetros:

  • x1, y1: Coordenadas del punto inicial.
  • x2, y2: Coordenadas del punto objetivo.

Descripción: Calcula el ángulo (en grados) entre dos puntos usando Math.atan2.

Retorna: El ángulo en grados.

point_spriteA_in_direction_spriteB(a_x, a_y, b_x, b_y)

Parámetros:

  • a_x, a_y: Coordenadas del sprite actual (ignoradas, ya que el sprite se obtiene por nombre).
  • b_x, b_y: Coordenadas del sprite objetivo (ignoradas, ya que el sprite se obtiene por nombre).

Descripción: Rota el sprite actual para que mire hacia el sprite objetivo usando sus posiciones actuales.

foward(distance)

Parámetros:

  • distance: Distancia para mover el sprite hacia adelante.

Descripción: Mueve el sprite actual hacia adelante la distancia especificada en la dirección que está mirando, actualizando las posiciones de los marcos en consecuencia.

distance_between_sprite_and_target()

Parámetros: Ninguno

Descripción: Calcula la distancia euclidiana entre el sprite actual y el sprite objetivo.

Retorna: La distancia como un número.

set_x_to(X)

Parámetros:

  • X: La coordenada x para establecer en el sprite.

Descripción: Establece la coordenada x del sprite actual y actualiza las posiciones de sus marcos.

set_y_to(Y)

Parámetros:

  • Y: La coordenada y para establecer en el sprite.

Descripción: Establece la coordenada y del sprite actual y actualiza las posiciones de sus marcos.

turn_right(degrees)

Parámetros:

  • degrees: Número de grados para rotar el sprite en el sentido de las agujas del reloj.

Descripción: Rota el sprite actual en el sentido de las agujas del reloj por los grados especificados y actualiza las posiciones de los marcos.

turn_left(degrees)

Parámetros:

  • degrees: Número de grados para rotar el sprite en sentido contrario a las agujas del reloj.

Descripción: Rota el sprite actual en sentido contrario a las agujas del reloj por los grados especificados y actualiza las posiciones de los marcos.

get_x_value()

Parámetros: Ninguno

Descripción: Recupera la coordenada x del sprite actual.

Retorna: La coordenada x como un número.

get_y_value()

Parámetros: Ninguno

Descripción: Recupera la coordenada y del sprite actual.

Retorna: La coordenada y como un número.

sprite_width()

Parámetros: Ninguno

Descripción: Recupera el ancho del sprite actual.

Retorna: El ancho como un número.

sprite_height()

Parámetros: Ninguno

Descripción: Recupera la altura del sprite actual.

Retorna: La altura como un número.

biased(a, b, c, d)

Parámetros:

  • a, b: Límites del rango para el primer intervalo.
  • c, d: Límites del rango para el segundo intervalo.

Descripción: Verifica si el punto c está dentro del rango [a, b) o si d está dentro de (a, b].

Retorna: Booleano que indica si se cumple la condición.

unbiased(a, b, c, d)

Parámetros:

  • a, b: Límites del rango para el primer intervalo.
  • c, d: Límites del rango para el segundo intervalo.

Descripción: Verifica si alguno de los intervalos se superpone con el otro usando la función biased.

Retorna: Booleano que indica superposición.

collision(r1, r2)

Parámetros:

  • r1, r2: Objetos rectangulares con propiedades top, bottom, left, right.

Descripción: Verifica si dos rectángulos se superponen usando la función unbiased para sus rangos x e y.

Retorna: Booleano que indica si los rectángulos colisionan.

intersectRect(s1, s2)

Parámetros:

  • s1, s2: Objetos sprite con propiedades x, y, width, height.

Descripción: Convierte las coordenadas y dimensiones de los sprites en objetos rectangulares y verifica la colisión usando la función collision.

Retorna: Booleano que indica si los cuadros delimitadores de los sprites se intersectan.

current_sprite_clones_are_colliding_with_target_sprite_clones()

Parámetros: Ninguno

Descripción: Verifica si el sprite actual (o su clon) colisiona con el sprite objetivo (o su clon) usando detección de colisión de cuadros delimitadores.

Retorna: Booleano que indica colisión.

is_colliding_with_target()

Parámetros: Ninguno

Descripción: Verifica si el sprite actual colisiona con el sprite objetivo usando detección de colisión de cuadros delimitadores.

Retorna: Booleano que indica colisión.

next_fame()

Parámetros: Ninguno

Descripción: Avanza el sprite actual al siguiente marco en su animación, volviendo al primer marco si está al final. Actualiza la visibilidad según el estado oculto del sprite.

set_frame_to_frame_number(frame_number_or_alias, force_game_object)

Parámetros:

  • frame_number_or_alias: Número de marco o alias a establecer.
  • force_game_object: Nombre de sprite opcional para sobrescribir el sprite actual.

Descripción: Establece el marco del sprite al número o alias especificado, manejando tanto sprites clonados como no clonados. Actualiza la visibilidad en consecuencia.

current_frame_number()

Parámetros: Ninguno

Descripción: Recupera el número de marco actual del sprite. Oculta el marco actual antes de retornar el número.

Retorna: El número de marco actual o 0 si no existen marcos.

update_position_of_frames(game_Object)

Parámetros:

  • game_Object: El objeto sprite cuyos marcos necesitan actualizarse.

Descripción: Actualiza la posición y rotación de todos los marcos de un sprite para que coincidan con la posición y ángulo del sprite principal.

loadAndPlaySound(key, url)

Parámetros:

  • key: La clave para el activo de audio.
  • url: La URL del archivo de audio a cargar.

Descripción: Carga un archivo de audio si no está ya en caché y lo reproduce usando el sistema de audio de Phaser.
